add

  • add(selector: string, bindings: IKeyBinding[]): IDisposable
    • Defined in manager.ts:107

    Add key bindings to the key map.

    Parameters

    • selector: string

      The CSS selector for the key bindings.

    • bindings: IKeyBinding[]

      The key bindings to add to the key map.

    Returns IDisposable

    A disposable which will remove the key bindings.

    Notes

    If the selector is an invalid CSS selector, a warning will be logged to the console and undefined will be returned.

    If the key sequence for a binding is invalid, or if a binding has a null handler, a warning will be logged to the console and that binding will be ignored.

processKeydownEvent

  • processKeydownEvent(event: KeyboardEvent): void
    • Defined in manager.ts:160

    Process a 'keydown' event and invoke the matching key bindings.

    Parameters

    • event: KeyboardEvent

      The event object for a 'keydown' event.

      Notes

      This should be called by user code in response to a 'keydown' event. The keymap does not install its own event listeners, which allows user code full control over the nodes for which the keymap processes events.

    Returns void
